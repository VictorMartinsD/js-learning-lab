# 🧪 JS Learning Lab

Bem-vindo ao meu laboratório pessoal de estudos em JavaScript! Este repositório é o registro prático da minha jornada de domínio da linguagem, focado em fundamentos sólidos, manipulação do DOM e lógica de programação.

## 🧠 Metodologia: Repetição Espaçada & Análise de Erros

Diferente de apenas copiar códigos, utilizo um sistema de **Repetição Espaçada (Spaced Repetition)** para garantir a retenção do conhecimento.

- **Ciclo de Revisão:** Cada exercício é praticado no dia atual, 1 dia depois, 1 semana e 1 mês após o primeiro contato.
- **Log de Evolução:** Não apago meus erros. Eu os documento para entender o "porquê" da falha, transformando lacunas de conhecimento em aprendizado consciente.

> [!TIP]
> Todo o meu progresso teórico, resumos de aulas, gabaritos e **registros detalhados de erros** estão centralizados no meu Notion.

## 📂 Estrutura do Repositório

A organização do repositório é modular e segue a ordem cronológica dos meus estudos dentro da pasta `.aulas`. Cada diretório interno é numerado e nomeado conforme o tema abordado (ex: `.aulas/01-logica`, `.aulas/02-funcoes`), permitindo que a própria lista de arquivos sirva como o índice de conhecimentos adquiridos.

---

## 🛠️ Como utilizo este Lab

Sempre que inicio um novo tema, utilizo a pasta `.aulas/00-template` para gerar a estrutura base do exercício. Se eu cometo um erro de sintaxe ou lógica, eu anoto o erro no Notion antes de aplicar a correção aqui. Isso garante que eu não apenas corrija o código, mas aprenda com a falha para evitar repetições.
